# The License

**Source-available licensing for people who love open source enough to want it to survive.**

Open source is a miracle. It's also a communal garden where sometimes nobody brings the water, everyone picks the vegetables, and eventually the weeds take over. This isn't malice — it's **entropy**.

The License is a practical alternative: share your code, protect your work, keep the project sustainable. No flame wars, no moral panic, just a license that says "you can look, you can learn, you can use — but let's not pretend everyone's self-interest always aligns with the commons."

### What's in here

- **`LICENSE.md`** — The actual license (AWFixer Source Available License v0.4). Read it. Argue with it. Fork it if you want.
- **`website/`** — The project's website, built with Astro + React + Tailwind + shadcn/ui.

### The vibe

We're not mad. We're not cynical. We just think pretending the commons is always self-sustaining doesn't help anyone. Bring your best ideas, your worst takes, and a willingness to be wrong.

Pull requests welcome. Eye rolls also welcome but less useful.
